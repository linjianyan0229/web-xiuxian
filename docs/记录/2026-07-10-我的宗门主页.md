# 我的宗门主页（/my-sect）

- 日期：2026-07-10 21:03
- 需求：玩家加入宗门后，「宗门」入口应直达**本宗门的界面**（背景图 `web/image/zongmenbeijing.webp`），并把宗门的按钮、交互功能装上（可分期完成）。

## 改动内容

- 后端：无（全部复用宗门一期接口：详情/门人名录/任免/逐出/传位/修订资料/退出/解散）。
- 前端：
  - 新路由页 `views/MySectView.vue`（`/my-sect`，requiresAuth，已入 `GAME_PAGES` 获滑动过渡）：`zongmenbeijing.webp` 全屏背景 + 宣纸轻纱，与 HUD 同规锁一屏内滚、窄屏放开单列。
    - **顶栏**：返回洞府、宗门名大标题、我的职位徽章、灵石。
    - **左栏信息卡**：宗徽/名称/入门要求 + 宗主（道号/境界）/门人数/活跃度/立派与入宗时间 + 简介；操作按钮——门人名录（复用 `SectMembersModal`，含任免/逐出/传位）、修订资料（rank≤3，复用 `SectCreateModal` 编辑模式）、天下宗门（跳 `/sect`）、退出宗门 / 解散宗门（宗主）。
    - **右栏设施宫格**（二期占位，对应职位权限文稿 §7/§8）：藏经阁/宗门商店/闭关室/宗门仓库/宗门任务/执法堂/宗门大牢/山峰，8 卡带「待启」印，点击 toast 敬请期待。
    - **归属兜底**：进页无 `sect_id` 或详情 `my` 为空（被逐/解散/退宗残留）→ 同步 profile 后重定向 `/sect`；人事变动（`changed`）后刷新 profile+详情（宗主可能易位、我可能被改任）。
  - 分流与跳转：首页导航「宗门」按 `auth.user.sect_id` 分流（有宗 → `/my-sect`，散修 → `/sect`）；宗门列表页顶栏加「回本宗 ›」按钮（有宗时显示）；**拜入成功**（详情弹窗 `joined` 事件）与**立派成功**均直接进入 `/my-sect`。
  - `App.vue` 的 `GAME_PAGES` 增加 `my-sect`。
