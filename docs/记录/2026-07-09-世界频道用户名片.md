# 世界频道用户名片

- 日期：2026-07-09 00:31
- 需求：世界频道里点击某位道友，弹出其部分公开信息卡片——展示名称、头像、所属宗门，并以性别对应立绘为背景。

## 改动内容
- 后端：`world_messages` 消息视图 `MESSAGE_SELECT` 增加 `LEFT JOIN sects` 带出 `sect_name`（原已带 dao_name/avatar/gender/realm_name），使频道消息项自带宗门名，前端点击即用无需额外请求。
- 前端：新增频道用户名片弹窗 `ChatUserModal.vue`（性别立绘 boy.webp/girl.webp 作背景 + 头像 + 道号 + 境界标签 + 性别/所属宗门两行）；`HomeView.vue` 世界频道消息的头像与道号可点击打开名片，加手型光标与悬停高亮。

## 涉及文件
- 新增：`web/src/components/ChatUserModal.vue`（频道用户公开信息名片弹窗）
- 修改：`node/src/models/worldMessageModel.js`（MESSAGE_SELECT 增 sects join 与 sect_name）
- 修改：`web/src/views/HomeView.vue`（导入并挂载 ChatUserModal，消息头像/道号绑点击 openChatUser，样式加 cursor/hover）

## 验证
- 前端 `npm run build` 通过（431ms，无报错）。
- 数据链路：消息视图三处查询（listRecent/listAfter/findById）共用 MESSAGE_SELECT，均自动带出 sect_name；散修（sect_id 为 NULL）显示「散修」，性别缺省按男修立绘。

## 遗留
- 名片仅展示消息项已有字段（名称/头像/境界/性别/宗门），未拉取灵石/修为等完整属性；如需更多信息需另开公开用户信息接口。
